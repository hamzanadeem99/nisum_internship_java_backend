# Nisum Internship Projects

Backend development projects completed during my 
3-month Java Developer Internship at **Nisum Pakistan** 
— a global technology company operating across 10+ countries.

## Projects

### 1. Shopping Cart Microservices System
A 2-microservice Spring Boot application for managing 
shopping carts and items.
- REST APIs for cart and item operations (CRUD)
- JPA + MySQL for database persistence
- Grand total and discount calculation
- Exception handling, input validation
- Unit test coverage above 80%

**Tech:** Java · Spring Boot · MySQL · JPA · REST API · Git

---

### 2. Electronics Warehouse Management System
A console-based Java application for managing an 
electronics warehouse with laptops, phones, and accessories.
- Full CRUD operations with MySQL/JDBC
- Multi-threaded CSV file import
- Role-based authentication system
- Input validation and error handling

**Tech:** Java · MySQL · JDBC · Multithreading · Git
